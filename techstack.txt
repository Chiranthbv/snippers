Frontend Tech Stack — Next.jsThe frontend is a separate project that calls the Spring Boot API. It is built with Next.js, styled with Tailwind CSS, and animated with Framer Motion — a modern SaaS stack used by companies like Vercel, Linear, and Stripe.TechnologyVersionPurposeNext.js14.x (App Router)React framework — handles routing, SSR, SSGReact18.xUI library — component-based UITypeScript5.xType safety — catches bugs before runtimeTailwind CSS3.xUtility-first styling — no separate CSS files neededFramer MotionLatestSmooth animations, page transitions, hover effectsAxios / fetchBuilt-inMakes HTTP requests to the Spring Boot APIshadcn/uiLatestPre-built accessible components (buttons, modals, etc.)Lucide IconsLatestClean SVG icon librarynext-themesLatestDark mode toggle supportPrism.js / ShikiLatestSyntax highlighting for code snippetsReact Hot ToastLatestToast notifications (success, error messages)Node.js20 LTSRuntime to run the Next.js dev servernpm / pnpmLatestPackage manager — like Maven but for frontendWhy Next.js over plain React?✓ File-based routing — no react-router setup needed✓ Server-side rendering (SSR) — snippets load fast, good for SEO✓ API routes — can add small serverless functions if needed✓ App Router (Next.js 14) — cleaner layouts and loading states✓ Image optimization built-in✓ Deploys to Vercel for free in 30 seconds5. Frontend ↔ Backend ConnectionEnvironment Variables (Frontend).env.local in the Next.js projectNEXT_PUBLIC_API_URL=http://localhost:8080   # local devNEXT_PUBLIC_API_URL=https://your-api.railway.app  # productionExample API Call (Next.js → Spring Boot)Fetching snippets from the Java API// In a Next.js component or server action:const res = await fetch(`${process.env.NEXT_PUBLIC_API_URL}/api/snippets`);const data = await res.json();// Authenticated request (send JWT token):const res = await fetch(`${process.env.NEXT_PUBLIC_API_URL}/api/snippets`, {  headers: { Authorization: `Bearer ${token}` }});7.
